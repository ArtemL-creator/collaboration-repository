Фаза 1: 
В первой фазе мы создаем программу для подсчета колличества слов в файле text.txt с выводом в консоль.
Программа находится в файле Main2.java

Как работает программа:
Данная программа считает количество слов в файле "text.txt", который находится в директории "src/main/resources" в проекте на языке Java.

Первым делом в методе main создается объект класса File,
который представляет указанный файл. Затем создается объект класса BufferedReader,
который используется для чтения содержимого файла. FileReader используется для чтения содержимого файла.

Далее происходит чтение каждой строки файла с помощью метода readLine() объекта reader. 
Внутри цикла while выполняется проверка, готов ли объект reader к чтению следующей строки с помощью метода ready().

Для каждой строки файла выполняется следующий алгоритм:

-Создается переменная count для хранения количества слов в строке.
-Создается флаг flag, который используется для определения начала слова.
-Для каждого символа в строке проверяется, является ли он пробелом или нет.
-Если символ не является пробелом и флаг равен true, то это начало нового слова и увеличивается счетчик слов count на 1, а флаг flag устанавливается в false.
-Если символ является пробелом, то флаг flag устанавливается в true.
-По окончанию строки, количество слов в ней добавляется к общему количеству слов count.

После того, как весь файл был прочитан, объект reader закрывается с помощью метода close(). 
И программа выводит общее количество слов в файле в консоль с помощью метода println().

Таким образом мы имеем программу для подсчета слов в файле заданном в коде программы с выводом результата в консоль.

Фаза 2:
Вторая фаза заключается в создании более продвинутой версии программы.

Новая версия программы является графическим приложением, написанным с использованием библиотеки Swing. 
Она предназначена для перетаскивания текстовых файлов и подсчета количества слов в выбранном файле. 
Приложение содержит окно JFrame, текстовую метку JLabel, 
объект DropTarget для перетаскивания файлов и метод countWords, который подсчитывает количество слов в выбранном файле.

Версия программы в первой фазе также подсчитывает количество слов в текстовом файле, 
но она написана без использования графического интерфейса.

Таким образом, основное отличие между двумя версиями программамы заключается в том, 
что первая выполняется в консоли и читает конкретный файл, заданный в коде программы. 
а вторая является графическим приложением, которое позволяет пользователю выбирать файлы для подсчета слов.

В третьей фазе было добавлено ограничение. 

Ограничение заключается в том, что слова долны начинаться либо с латиницы, либо с киррилицы. Т.е. теперь программа не считает за слова цифры и другие символы, не относящиеся к буквам двух этих алфавитов.
